## 🎯 Sobre o Projeto

English Flow revoluciona o ensino de inglês através de:

- **IA Conversacional**: Pratique diálogos reais 24/7
- **Gamificação Total**: XP, badges, streaks, ranking
- **Repetição Espaçada**: Algoritmo inteligente de revisão
- **5.000+ Frases Práticas**: Foco em conversação real
- **Preço Justo**: R$ 39,90/mês (10x mais barato)
- **Flexibilidade**: Estude onde e quando quiser

## 🛠️ Tecnologias

**Frontend:** React 18, TypeScript, Vite, Tailwind CSS, Zustand, React Query

**Backend:** Node.js, Express, TypeScript, Prisma, PostgreSQL

**Infraestrutura:** Vercel, Railway, Supabase

## ✨ Funcionalidades

- ✅ Sistema de autenticação completo (JWT)
- ✅ 100 frases do Level 1
- ✅ 10 categorias de lições
- ✅ Sistema de progresso com XP
- ✅ Dashboard com estatísticas
- ✅ Sistema de conquistas
- ✅ Design responsivo
