# Day 41 - Multi-language (i18n)

Complete internationalization system supporting 10+ languages.

## Features
- Dynamic language switching
- Translation keys system
- RTL support (Arabic, Hebrew)
- Locale-based date/number formatting
- User language preferences
- Auto-detect browser language

## Supported Languages
1. Portuguese (pt-BR) - Default
2. English (en-US)
3. Spanish (es-ES)
4. French (fr-FR)
5. German (de-DE)
6. Italian (it-IT)
7. Japanese (ja-JP)
8. Chinese (zh-CN)
9. Korean (ko-KR)
10. Russian (ru-RU)

## Usage
```javascript
import { t } from './i18n'
console.log(t('app.welcome', 'en-US')) // "Welcome to English Flow"
```
